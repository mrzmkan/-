# ئارشیڤی ئیجازاتی زانای کوردی
## Kurdish Scholar Ijazah Archive — GitHub Pages Site

وێبسایتی کوردی بۆ نیشاندانی ئیجازات و مستەنداتی زانای ئیسلامی، لەسەر GitHub Pages.

---

## 📁 پڕۆژەکە چۆن دادەمەزرێت

### ١. فایلەکان بنێ لە ڕیپۆزتری GitHub

ئەم فایلانە هەموویان بنێ لە ڕووتی ڕیپۆزتری (`/`):

```
index.html      ← مالپەڕی گشتی
admin.html      ← پەنەلی بەڕێوەبردن
data.json       ← هەموو داتاکان لێرەدایە
README.md       ← ئەم فایلە
```

---

### ٢. GitHub Pages چالاک بکە

١. بچۆ بۆ ڕیپۆزتریەکەت → **Settings** → **Pages**
٢. لە **Source** دا `main` branch هەڵبژێرە
٣. **Save** دابگرە
٤. چەند خولەکێک لەدوا مالپەڕەکەت ئامادەیە لە:
   `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

---

### ٣. دروستکردنی GitHub Personal Access Token (PAT)

**ئەمەی پێویستە بۆ ئەوەی پەنەلی ئەدمین بتوانێ داتا بنووسێت.**

١. بچۆ بۆ: https://github.com/settings/tokens/new
٢. **Note:** هر ناوێک بنووسە، بۆ نمونە `ijazah-admin`
٣. **Expiration:** هەڵبژێرە
٤. **Select scopes:** تیک لە **`repo`** بکەوە (هەموو ژێر-خانەکانی خۆکارانە تیک دەبن)
٥. **Generate token** دابگرە
٦. تۆکێنەکە — کە بە `ghp_` دەستپێدەکات — کۆپی بکە
   ⚠️ **تۆکێن تەنها یەکجار نیشاندەدرێت!**

---

### ٤. ئەدمین پەنەل رێکبخە

١. بچۆ بۆ `admin.html`
٢. چوون بۆ **ڕێکخستنەکان** (خۆکارانە دەکرێتەوە ئەگەر تازەیە)
٣. ئەم زانیاریانەی خوارەوە پرکەوە:
   - **GitHub Username**: ناوی بەکارهێنەرت لەسەر GitHub
   - **Repository Name**: ناوی ڕیپۆزتری
   - **Data File**: `data.json` (ئەمەی بگۆڕنەوە ئەگەر ناوی دیکەیەتی)
   - **Personal Access Token**: تۆکێنەکەی کۆپیکرد
٤. **پاشەکەوتکردن** دابگرە
٥. **تاقیکردنەوەی بەستەر** دابگرە بۆ دڵنیابوون

---

### ٥. رووتی `index.html` رێکبخە (پێویستە)

وا کە `index.html` ئاگاداری ڕیپۆزتریەکەت بێت،
سەر لای **کۆدی JavaScript** ئەم دوو خانەیە دەستکاری بکە:

```javascript
const REPO_OWNER = 'YOUR_GITHUB_USERNAME';   // ← ناوی بەکارهێنەر
const REPO_NAME  = 'YOUR_REPO_NAME';         // ← ناوی ڕیپۆزتری
```

ئەگەر بەبێ تۆکێن بێت، مالپەڕی گشتی بە ئاشکرا داتا دەخوێنێتەوە لە GitHub API بەبێ پشتڕاستکردنەوە.

---

## 🔐 ئەمنیەتی تۆکێن

- تۆکێنەکەت تەنها لە `sessionStorage` بەکارهێنراوە — ئەمەش تاکو تابەکە داخرا دەمێنێتەوە
- تۆکێنەکە هەرگیز بۆ فایلی داتا نادرێت و commit ناکرێت
- هەر ئەدمینێک پاش خۆشوێنبوونەوە دەبێت تۆکێن دووبارە داخڵ بکات
- ئەگەر تۆکێن لەبیر چوویت، لە GitHub دووبارەی دروستبکەرەوە

---

## ✏️ بەرێوەبردنی داتا

| کار | شوێن |
|-----|------|
| زیادکردنی ئیجازەی نوێ | Admin → ئیجازاتەکان → **زیادکردنی ئیجازە** |
| دەستکاریکردن | Admin → ستونی کردارەکان → **✏️ دەستکاری** |
| سڕینەوە | Admin → ستونی کردارەکان → **🗑️ سڕینەوە** |
| گۆڕانی پرۆفایل | Admin → **پرۆفایلی زانا** |

هەموو گۆڕانکارییەک خۆکارانە `data.json` لەناو GitHub نوێدەکاتەوە.
